Rough [Nim](https://nim-lang.org/) compiler autocomplete generator for [carapace](https://github.com/carapace-sh/carapace)

## Usage

- [Install](https://carapace-sh.github.io/carapace-bin/install.html) and [setup](https://carapace-sh.github.io/carapace-bin/setup.html) carapace for your shell, by official instructions
- Add [nim.yaml](https://github.com/KawaiiSelbst/carapace-nim-spec-gen/releases/download/0/nim.yaml) from releases page to your ~/.config/carapace/specs
